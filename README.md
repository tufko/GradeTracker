Project: Grade Manager

Description:
This program manages student grades. It stores multiple grades per student, calculates averages, and assigns letter grades. It prints a full class report.

How to run:

Open project in Java IDE
Run Main.java

Features:

Add students
Store multiple grades per student
Calculate average score
Assign letter grades
Display class report

Concepts used:

variables and data types
if statements
loops
methods
ArrayList
classes and objects
exception handling

Reflection:
This project helped me understand how to organize a Java program using multiple classes. I improved my ability to debug errors related to file names and class structure. I also learned how GitHub is used to store and submit code.
